# Agent Editable Video Pipeline Kit

Turning video editing into transcript + JSON + code makes content production inspectable, repeatable, and agent-improvable instead of a one-off opaque timeline.

Inspired by: https://x.com/trq212/status/2064826394589442448

## What it contains

- `artifact-package/` — Obsidian package note mirrored from x-intel.
- `prototype/` — concrete trial packet / checklist.
- `infographic/` — workflow explainer when available.
- `skill-draft/` — reusable procedure draft when available.
- `improvement-loop/` — recurring improvement tracker.

## Quickstart

Open the prototype markdown file and fill it during a real trial. This repo intentionally ships templates and gates, not fabricated validation results.
