# Agent Editable Video Pipeline Kit Prototype

Source: [[Agent-Edited Video as Text and Code Pipeline]]

## What it demonstrates

Turning video editing into transcript + JSON + code makes content production inspectable, repeatable, and agent-improvable instead of a one-off opaque timeline.

## How to use

Open `One-Minute Agent Video Trial Packet.md` and fill it during a real trial. Then use `Video Evidence Attachment Manifest.md` to bundle the source metadata, transcript, JSON EDL, commands, render log, draft MP4, and verifier notes. After evidence is attached, use `Post-Trial Promotion Decision Card.md` to choose promote / pilot-only / iterate / hold. Keep evidence links in the packet; do not mark validation complete until artifacts are attached.
